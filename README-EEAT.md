## 📦 Installation

### Method 1: WordPress Admin (Recommended)

1. **Download** the plugin ZIP file
2. Go to **WordPress Admin → Plugins → Add New**
3. Click **Upload Plugin**
4. Choose the ZIP file
5. Click **Install Now**
6. Click **Activate Plugin**
7. ✅ E-E-A-T database tables created automatically!

### Method 2: Manual Upload

1. **Unzip** the plugin file
2. **Upload** the `delice-recipe-manager` folder to `/wp-content/plugins/`
3. Go to **Plugins** page in WordPress admin
4. **Activate** "WP Delicious Recipe"

## 🚀 Quick Start (5 Minutes)

### Step 1: Verify Installation
After activation, go to **Delice Recipes** menu and verify you see:
- ✅ E-E-A-T Features
- ✅ Author Profiles
- ✅ Recipe Testing
- ✅ User Submissions

### Step 2: Configure Display Settings
1. Go to **Delice Recipes → E-E-A-T Features**
2. Enable the features you want:
   - ☑️ Show Experience Features
   - ☑️ Show Expertise Features
   - ☑️ Show Authority Features
   - ☑️ Show Trust Features
3. Click **Save Settings**

### Step 3: Set Up Your Author Profile
1. Go to **Delice Recipes → Author Profiles**
2. Select your name from dropdown
3. Fill in:
   - Bio (200-300 words)
   - Experience years
   - Credentials
   - Specializations
4. Check "Verified" if you're a professional chef
5. Click **Save Profile**

### Step 4: Add E-E-A-T Data to a Recipe
1. **Edit any recipe**
2. In **"🧪 Recipe Testing"** sidebar box:
   - ☑️ Recipe has been tested
   - Enter number of tests
3. In **"🛡️ Safety & Trust"** box:
   - Check applicable allergens
   - Check dietary tags
   - Add food safety notes
4. Click **Update**
5. **View the recipe** on frontend to see E-E-A-T features!

## 🐛 Troubleshooting

### E-E-A-T Features Not Showing?

**Solution:**
1. Go to **Delice Recipes → E-E-A-T Features**
2. Enable display options
3. Click **Save Settings**
4. Clear site cache
5. Hard refresh recipe page (Ctrl+Shift+R)

### Database Tables Not Created?

**Solution:**
1. Deactivate plugin
2. Reactivate plugin
3. Tables will be created automatically

### CSS Styles Not Loading?

**Solution:**
1. Hard refresh browser (Ctrl+Shift+R)
2. Clear cache plugin
3. Check file permissions

For more troubleshooting, see **EEAT-INSTALLATION-GUIDE.md**

## 🐛 TROUBLESHOOTING

### Issue: New Menu Items Not Showing

**Solution:**
1. Deactivate and reactivate plugin
2. Clear WordPress cache
3. Go to Settings → Permalinks → Save
4. Log out and log back in
5. Hard refresh browser (Ctrl+Shift+R)

### Issue: Features Not Displaying on Frontend

**Solution:**
1. Check E-E-A-T Features → Display Settings
2. Ensure checkboxes are enabled
3. Clear site cache (if using cache plugin)
4. Hard refresh recipe page
5. Check meta boxes are filled in recipe edit screen

### Issue: CSS Not Loading

**Solution:**
1. Verify files exist:
   - `/admin/css/delice-eeat-admin.css`
   - `/public/css/components/delice-eeat.css`
2. Check file permissions (should be 644)
3. Hard refresh browser
4. Clear cache plugin
5. Deactivate conflicting plugins

### Issue: Database Tables Not Created

**Solution:**
Run in WordPress admin (Plugins → Add New → "Run Code" plugin):
```php
require_once WP_PLUGIN_DIR . '/delice-recipe-manager/includes/class-delice-recipe-eeat.php';
$eeat = Delice_Recipe_EEAT::get_instance();
$eeat->create_tables();
echo "Tables created successfully!";
```

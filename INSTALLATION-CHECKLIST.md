## Installation Steps

### Step 1: Upload & Activate
- [ ] Downloaded the ZIP file
- [ ] Uploaded to WordPress (Plugins → Add New → Upload)
- [ ] Clicked "Install Now"
- [ ] Clicked "Activate Plugin"

### Step 2: Verify E-E-A-T Features
Go to **Delice Recipes** menu and check for:
- [ ] ✅ E-E-A-T Features menu item visible
- [ ] ✅ Author Profiles menu item visible
- [ ] ✅ Recipe Testing menu item visible
- [ ] ✅ User Submissions menu item visible

### Step 3: Configure Display Settings
- [ ] Went to Delice Recipes → E-E-A-T Features
- [ ] Enabled "Show Experience Features"
- [ ] Enabled "Show Expertise Features"
- [ ] Enabled "Show Authority Features"
- [ ] Enabled "Show Trust Features"
- [ ] Clicked "Save Settings"

### Step 4: Set Up Author Profile
- [ ] Went to Delice Recipes → Author Profiles
- [ ] Selected my name from dropdown
- [ ] Added professional photo
- [ ] Wrote 200-300 word bio
- [ ] Added years of experience
- [ ] Added credentials (minimum 3)
- [ ] Added specializations
- [ ] Checked "Verified" (if professional chef)
- [ ] Clicked "Save Profile"

### Step 5: Test on a Recipe
- [ ] Edited an existing recipe
- [ ] Found "🧪 Recipe Testing" meta box in sidebar
- [ ] Checked "Recipe has been tested"
- [ ] Entered number of tests (e.g., 5)

- [ ] Selected applicable allergens
- [ ] Selected dietary tags
- [ ] Added food safety notes
- [ ] Clicked "Update"

- [ ] Verified testing badge displays
- [ ] Verified author box displays
- [ ] Verified safety information displays

If features aren't showing:

- [ ] Checked display settings are enabled
- [ ] Deactivated and reactivated plugin
